# Brightwell Carter & Lane LLP — Billing Portal (Demo)

A **visual demo clone** of a law-firm owner billing portal. Front-end only:
sign in with **any username and password** and click through every section
(A/R aging, invoices, collections, originating attorneys, P&L, partner
distributions, cash & balance sheet, A/P, WIP, top clients, Bill's Corner, etc.).

- All firm / client / attorney / partner names are **fictional**, and all
  dollar figures are **synthetic** (scaled sample data). Nothing maps to a real
  firm, person, or matter.
- Write/sync functionality (uploads, comments, documents, GitHub) is disabled —
  this is a look-and-feel demo only. Data is read from the bundled `data.json`.

## Run locally

```bash
python3 -m http.server 8753
# open http://localhost:8753
```
